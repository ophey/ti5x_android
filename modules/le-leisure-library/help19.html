Using this program you may print any of the following patterns in any quarter of the tape. These patterns are illustrated using an asterisk, however, you may choose any character, number or symbol that the PC-100A is capable of printing.

  • Initialize program : SBR CLR. 0 is Displayed. If calculator has fixed-decimal display format press 2nd Fix 9.
  • Select the patter you desire and note its number (see above)
  • Choose a character from the following chart and determine its code number. Each printed character is represented by a two-digit, row-column address code according to the following table. For example, A is code 13.
    x0x1x2x3x4x5x6x7

  • Enter the information from both steps above in the for Pattern#.Symbol#.
  • Select the quarter of the tape you wish to use by pressing the appropriate user-defined key.
    • A - First Quarter
    • B - Second Quarter
    • C - Third Quarter
    • D - Fourth Quarter
  • Repeat steps above for each quarter as desired, the press E to print the entire line.
  • Repeat for each line.
Note: If you need more than one type of character in a specific quarter of the tape, you can fill the print buffer directly by using the special operation codes described in the User's Manual (Op 01-04). Pressing E will print the contents of the print buffer.
