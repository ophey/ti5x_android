This program calculate the adjusted exposure time required to compensate for the change in magnification when making enlargements of different sizes. After one perfect print has been produced from a negative at any magnification, the proper exposure can be subsequently be calculated for a print at any other magnification.

  • Initialize program : SBR CLR
  • Enter focal length of first lens, f1, press A if in millimeters or 2ndA' if in inches.
  • Enter first negative-to-image distance, D1, press B if in millimeters or 2ndB' if in inches.
  • Enter first exposure time, t1, and press C
  • Enter focal length of second lens, f2, press A if in millimeters or 2ndA' if in inches. This step can be omitted if f2 = f1.
  • Enter second negative-to-image distance, D2, press B if in millimeters or 2ndB' if in inches.
  • Press C to find second exposure time.
  • Press D to find magnification
