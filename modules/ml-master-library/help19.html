ML-19 — Annuities

First, initialize the program with 2nd E´.

Then, choose which type of computation to perform:

  • 2nd A´ — Sinking Fund
  • 2nd B´ — Annuity Due/FV
  • 2nd C´ — Ordinary Annuity/PV
  • 2nd D´ — Annuity Due/PV

Then, enter 4 of the following 5 quantities, in any order:

  • N A — enter number of periods
  • %I B — enter percent interest rate per period
  • PMT C — enter periodic payment amount
  • PV/FV D — enter present value or future value (depending on computation chosen)
  • BAL E — enter balloon payment (note: if this amount is 0, just skip its entry, don’t enter 0)

To compute the missing value, enter 0 before pressing the corresponding key, i.e.

  • 0 A — compute number of periods N
  • 0 B — compute percent interest rate per period %I
  • 0 C — compute payment per period
  • 0 D — compute present value or future value
  • 0 E — compute balloon payment

Formulas Used

In the following formulas, i = %I ÷ 100. In all cases, %I is computed using Newton-Raphson iteration.

Sinking Fund

FV = PMT × [(1 + i)N - 1] ÷ i

N = ln (FV × i ÷ PMT + 1) ÷ ln (1 + i)

PMT = FV × i ÷ [(1 + i)N - 1]

Annuity Due/FV

FV = PMT × (1 + i) × ((1 + i)N - 1) ÷ i

N = ln (FV × i ÷ PMT + 1 + i) ÷ ln (1 + i) - 1

PMT = FV ÷ [(1 + i) × ((1 + i)N - 1) ÷ i]

Ordinary Annuity/PV

PV = PMT × (1 - (1 + i)-N) ÷ i + BAL × (1 + i)-N

N = ln ((PMT - i × BAL) ÷ (PMT - i × PV )) ÷ ln (1 + i)

PMT = (PV - BAL × (1 + i)-N) ÷ ((1 - (1 + i)-N) ÷ i)

BAL = (PV - PMT × (1 - (1 + i)-N) ÷ i) ÷ (1 + i)-N

Annuity Due/PV

PV = PMT × (1 + i) × (1 - (1 + i)-N) ÷ i + BAL × (1 + i)-N

N = ln ((PMT × (1 + i) ÷ i - BAL) ÷ (PMT × (1 + i) ÷ i - PV)) ÷ ln (1 + i)

PMT = (PV - BAL × (1 + i)-N) ÷ ((1 + i) × (1 - (1 + i)-N) ÷ i)

BAL = (PV - PMT × (1 + i) × (1 - (1 + i)-N) ÷ i) ÷ (1 + i)-N

Registers used: 01-14.
