This program will compute the moving average of the n last entered values of an indefinite series of numbers.

  1. First, enter n, which must be a positive integer: n A.
  2. Then, enter each new data value in turn and press B; the moving average will be displayed immediately afterwards.

Repeat step 2 as many times as desired.

To start a new series, start again from step 1.

Registers used: 01-05+n.
