First, enter the number of rows in A, and the number of columns in A. Then enter the elements of the matrix A starting with column 1. (To correct an entry error in column j, simply enter j and press B. Then reenter the entire column using the R/S key)

For Matrix Addition:

  • 1 C
  • Then enter the elements of matrix B using the R/S key. To correct en entry error in column j, simply enter j and press C. Then reenter the entire column using the R/S key.
  • Enter λ1 D
  • Enter λ2 D (steps to be done in sequence even if λ1 and λ2 are equal to 1)
  • Calculate λ1 A + λ2 B CLR E
  • Display the elements of C starting with column 1, 1 2nd A', then press R/S to show c11, c21, ... cmn in sequence.

For Matrix Multiplication

  • Enter the elements of column j of matrix B starting with b1j. To correct an entry error for bij, enter i, press 2nd B' and reenter bij using the R/S key.
  • Calculate column j of matrix C, 2nd C'
  • Display the elements of column j of matrix C, starting with c1j: 1 2nd D' and press R/S. c1j .. cmj are displayed in sequence.
  • To display cij only : i 2nd D' R/S

Formulas Used

If C = λ1 A + λ2 B, then:

cij = λ1 aij + λ2 bij

If C = AB, then:

cij = ∑k=1..n aik bkj

Registers used: 01-07

Plus for addition substraction : 08 through 2mn+7

Plus for multiplication : 08 through mn+2n+7
