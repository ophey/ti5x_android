This program contains an assortment of subroutines which allow you to determine the status of the calculator.

  • Save calculator status, press A. Can be restored with 2nd A'.
  • Initializes calculator status, press 2nd A'
  • Flags set status, press B. 0 means no flag, 0.0 that only flag 0 is set. Other flags are indicated by number to the left, 742.0 means that flags 7, 4, 2, 0 are set.
  • Fix-decimal status, press 2nd B'. Display the number of decimals or 9 for floating point selection.
  • Partitioning status, press C. Values between 0 and 10. Always 10 on this implementation.
  • Printer connect status, press the flag number and 2nd C'. The flag will be set or unset depending on the presence or not of the printer.
  • Angular mode status, press D. Display 0 for degrees, -1 for radians and 1 for grads.
  • Open parentheses status, press E. Display the number of open parentheses.

Registers used: 00-01
