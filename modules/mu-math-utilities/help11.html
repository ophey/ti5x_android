MU-11 — Gamma/Factorial

The gamma function Γ(x) is a generalization of the factorial function x! to non-integer x. For integer x, the equality holds:

Γ(x) = (x - 1)!

for the usual definition of the factorial function:

0! = 1

x! = (x - 1)! × x for integer x > 0

Gamma and factorial both increase vary rapidly for modest increases in x; thus, it is often useful to work with the logarithm of the gamma and factorial functions, to allow for larger operands before numerical overflow sets in.

This program will compute Γ(x), ln Γ(x), x! and ln x! for real x. The valid domains for x are as follows (upper bounds are approximate):

FunctionDomain
Γ(x)x ∈ 1 .. 70
ln Γ(x)x ∈ 1 .. 4.5535879 1010
x!x ∈ 0 .. 69
ln x!x ∈ 0 .. 4.5535879 1010

In each case, first key in the number x, then press the button corresponding to the function desired, as follows:

A
Compute Γ(x).
2nd A'
Compute ln Γ(x).
B
Compute x!.
2nd B'
Compute ln x!.
C
Compute exact x! for integer x only.

Formulas Used

ln Γ(x) is computed from the following approximation:

ln Γ(x) ≅ (x - ½) ln x - x + ½ ln (2π) + Σ(m = 1 .. 5) (B2m ÷ (2m(2m - 1)x2m - 1))

where the first five B2m (Bernoulli numbers) are as follows:

B2 = 1/6
B4 = -1/30
B6 = 1/42
B8 = -1/30
B10 = 5/66

Registers used: 09-10.
