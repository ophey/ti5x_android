MU-18 — Differential Equations

Runge-Kutta methods are a particular set of numerical techniques used for solving inital-value problems. This program uses a fourth-order Runge-Kutta method to solve differential equations of the type y' = f(x,y) and y'' = f(x,y,y').

Numerical methods consist of finding approximate solutions at particular points x0, x1, ... xn where the difference between successive x-values is a constant, w (xn+1 - xn) = w). The value for w is chosen by the user with a smaller w generally yielding a more accurate solution. In this program, w is determined by the number of divisions, n, that the user chooses. Usually, w will be made smaller by increasing n.

To use this program, you must enter a subroutine (as 2nd A') which defines y' and y'' and the initial values for x0, y0 or x0, y0, y'0.

  • Select learn mode
  • Use A' as label, 2nd A'
  • Enter f(x,y) or f(x,y,y'), do not use = or CLR. Use register 10 for x, 11 for y and 12 for y'.
  • End with INV SBR
  • Leave learn mode
  • Select program, 2nd Pgm 1 8
  • Enter initial x, press A
  • Enter initial y, press B
  • Enter initial y' if using a second-order equation, press C
  • Enter number of division, press D
  • To solve y' = f(x,y), enter value if independent variable for which the solution is desired, press E
  • Or to solve y'' = f(x,y,y'), enter value if independent variable for which the solution is desired, press 2nd E'
  • Display slope at value of independent variable using x⇌t

Registers used: 9-19
