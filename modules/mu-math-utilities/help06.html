This program will quickly order a list of up to 99 numerical elements using an advanced technique known as the Shell sort. When used in conjunction with the PC-100A, the program will print the ordered list with each element accompanied by its ordinal number.

The sorted numbers are stored sequentially in the data registers beginning with R01.

  • Initialize by pressing E
  • Enter first element, press A
  • Enter next element, press A or R/S
  • To sort, press B
  • List sorted data or display first sorted element, C or R/S
  • Display next sorted element, D or R/S
  • Repeat step above until all factors are found (flashing one in the display).

Registers: 00 Pointer; 01 - 0n Data
