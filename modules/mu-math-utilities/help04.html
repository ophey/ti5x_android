MU-04 — Printer Formatting

Computer print statements which combine alpha messages and data on the same line can be simulated when this program is used in conjunction with Program MU-03, Alpha Messages. This is accomplished in three steps:

  • Store the alpha message, leaving space for data, using MU-03.
  • Enter data into the line using the format statement of this program.
  • Print the combined line.
  • Store alpha message using MU-03, leaving space for data.
  • Select program, 2nd Pgm 04
  • Enter data, press x⇌t to place it in T-register.
  • Enter format statement, for no leading zeros press A or with leading zeros press 2nd A'
  • Print line by pressing, B

The format statement is [skip][left].[right][line]

  • [skip], number of spaces data is intended from left side of tape.
  • [left], number of digits to left of decimal.
  • [right], number of decimal digits.
  • [line], line number (same as corresponding alpha message)

Registers: 00 - 07
