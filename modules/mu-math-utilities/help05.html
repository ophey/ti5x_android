Plotting and graphing functions, equations, and relationships is a useful tool for many different situation. The Superplotter program allows you to plot or graph up to 10 different functions simultaneously. For independent variable you only need to enter the starting value (X0), the increment (Δx), and the number of points (per function) you wish to print. For the dependent variable, enter the minimum value (ymin), the maximum value and the number of tapes desired. The output values will be properly scaled for printing with the horizontal increments being (ymax - ymin) / (20 x #Tapes).

On the tapes, alignment marks are printed each 10 vertical increments.

Each of the function you want to print must be defined by a series of keystrokes and entered as a subroutine. First function being 2nd A' and continuing up to E.

  • Enter subroutines for user-defined functions.
  • Select program, 2nd Pgm 05
  • Enter initial value x0, press A
  • Enter x increment, press B
  • The 3 following values must be entered in sequence:
    • Enter minimum y value, press C
    • Enter maximum y value, press D
    • Enter number of tapes desired, press E
  • Enter number of functions, press 2nd D'
  • Enter number of points to be plotted and print the graph by pressing 2nd E'

Registers: 08 - 25
