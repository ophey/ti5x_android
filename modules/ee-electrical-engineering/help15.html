Given the impulse response for a linear system, one can use the convolution integral to find the system's output for a specified input waveform. The program generates outputs at intervals of Δt using the trapezoidal rule.

The user specifies the integration step size (Δ) by the input of n0. Then Δ = Δt / n0. Note that &Delat; is held constant for each successive evaluation of y(t).

The user specifies the number of equi-spaced arguments (N) for which the integral is to be evaluated. After the output of N values of y(t), one may generate additional values (in increments of Δt)

The functions x(t) and h(t) are entered in program memory as a series of keystrokes.

  • Initialize program, RST

    Enter functions

  • Select learn mode, LRN
  • Use A' as the label for x(t) subroutine, enter 2nd Lbl and 2nd A'
  • Enter x(t) as a series of keystrokes. Assume t is in the display. Do not use = or CLR or any used registers.
  • End subroutine with INV SBR
  • Use B' as the label for h(t) subroutine, enter 2nd Lbl and 2nd B'
  • Enter h(t) as for x(t)
  • Exit learn mode, LRN

    Compute convolution

  • Select program 15, 2nd Pgm 1 5
  • Enter number of panels in each Δt (n0), press A
  • Number of values of y(t) desired (N), press B
  • Enter increment Δt, press C
  • Execute without printer, press 2nd E', display t1 and y(t1)

    press R/S N times to display the N values, display ti and y(ti)

  • Execute with printer, press E, print ti and y(ti)
  • If additional values of y(t) are desired without printer, 2nd D', display tN+1 and y(tN+1)
  • If additional values of y(t) are desired with printer, D, print tN+1 and y(tN+1)
