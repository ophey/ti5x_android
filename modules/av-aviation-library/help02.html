This program may be used to determine a flight plan for a trip of several legs with winds aloft. The wind triangle is solved using wind direction and velocity, magnetic compass variation, true airspeed, and true course to indicate magnetic heading and ground speed. The flying time, ETA, and fuel for each leg are computed, as are total time enroute and total fuel required.

The user should note that WD, WV, TAS, Burn and Var are assumed to be average and constant values for each leg, while in fact they may vary significantly and could seriously affect the accuracy of Fuel, GS (thus ETE), and MH.

Formulas Used

TH = TC + arcsin [ (WV/TAS) x sin (WD - TC) ]

MH = TH + Var

GS = TAS x cos (TH - TC) - WV x cos (WD - TC)
