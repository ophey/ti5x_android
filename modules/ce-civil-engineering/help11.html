STR 11 - Maximum Moment and Shear

In a beam, or beam segment, in which the uniform load Q≠0 and the moments MA, MB at the ends left and right are known, the program calculates:

  • the shear stress TA, TB at the ends;
  • the maximum bending moment MX;
  • the abscissa X0 in which T=0 and moment M=MX;
  • the abscissas X1, X2 in which M=0;
  • the areas SA, SB of the shear stress diagram to the left and right of the X0 coordinate.

The units of measurement must be consistents and those of the International System: lengths in meters, forces in Newton and related multiples or submultiples. If the lengths are measured in meters (m) and the forces in (dN), the bending moment must be measured in (dNm) and the distributed load Q in (dN/m).

The sign of the bending moment and shear stress must respect the convention of the Science of the Construction.

Instructions

The instructions for using the program are summarized in the "card emulation”, similar to the magnetic card of the TI-59, which appears below the display when the APP calls the program from the library module. It summarizes the instructions assigned to each definable key, transcribed in the box referring to the key.

1 - Provide the data: Each input procedure is independent.

  • enter MA (dNm), press  A : display= previously stored data.
  • enter MB (dNm), press  B : display= previously stored data.
  • enter L (m), press  C : display= previously stored data.
  • enter Q (dN/m), press  D : display= previously stored data.

To view a data, press its definable key twice. A first time to view it and correct it if necessary. A second time, to store it again in the same register.

2 - Results:

Pressing  E  the program will display the results TA.
Pressing in succession R/S wll display the others results TB, X0, SA, SB, MX, X1, X2.
The program will also print a report with data and results.
If M has a constant sign, the program, instead of X1 and X2, displays MX flashing and prints M CONST SIGN.
If M changes sign on the outside of the beam, there will be X1<0 and/or X2>L.
For a new calculation, reset only the data that changes.

Example 1

Beam data: MA=-4000 dNm, MB=-3000 dNm, L= 5.5 m, Q= 3000 dN/m;

  1. Set data:
    • MA: enter -4000 dNm, press A>. Display = previous data stored.
    • MB: enter -3000 dNm, press B. Display = previous data stored.
    • L: enter 5.5 m, press C. Display = previous data stored.
    • Q: enter 3000 dN/m, press D. Display = previous data stored.
  2. Results: press E. The program display TA = 8431.818182 dN.
    Pressing R/S in sequence will display the other results, in order:
    TB, SA, SB, X0, MX, X1, X2.
    The program will also print the following report:
