STR-01 — Geometric Characteristics of any Section

For any section delimited by a closed polygonal, possibly hollow, the program calculates area A, static moment with respect to the X axis “S/x”, Moment of inertia with respect to the X axis “J/x”, order of center of gravity “YG” and moment of inertia with respect to a barycentric axis parallel to the X axis “J/G”.

The reference system is set by the operator.

Description of the method

The vertices of the external polygon must be numbered progressively clockwise and, for hollow sections, the vertices of the internal polygon must be numbered progressively counterclockwise.

The coordinates of the vertex i of the external polygon and the coordinates of the vertex i' of the internal polygon are indicated respectively with xi, yi and xi’ and yi’.

If a definable key is assigned the input function of a single numeric data, for example Xi, only the description Xi>" is shown to indicate to type first the value of Xi and then, with the symbol ">" to press the definable key. If a double input is assigned to the same key, Xi > Yi :" will be written to indicate to the operator to first type the numerical value of Xi, then, with the ">" symbol, to press the definable key; then type Yi", and finally, with the symbol ":", press R/S to resume processing.

When the ">" symbol is indicated first, for example "> Xi: Yi:", the operator must first press the definable key, then the numerical data Xi, then the R/S key followed by numeric value of Yi" and finally the R/S key again. This last procedure is used to perform "output-input" operations of the numerical data, which allows the operator, before entering the value of the data to be stored, to view the value already stored on the display and, if necessary, correct it.

  1. Print option: 2nd St flg 1
  2. Initialization : 2nd E'
  3. Start external polygonal : 2nd A'
  4. Provide the coordinates of the vertices “1” of the external polygon, X1 A Y1 R/S
  5. Provide the coordinates of the vertices “2” of the external polygon, X2 A Y2 R/S
  6. continues to the last vertex “n”
  7. External polygon end : 2nd B'
  8. Start internal polygonal (if the section is full skip to step 9) : 2nd A'
  9. Provide the coordinates of the vertices “1” of the internal polygon, X1' A Y1' R/S
  10. continues to the last vertex “m”
  11. Ask for the results : E The values S/x J/x, Y(G) and J(G) will be displayed on the print or after each press to R/S if prinout not activated.

Example

External polygon: 

   1  2  3  4  5  6  7
x 10 60 60 50 50 20 10
y 50 50  0  0 10 40 40

Internal polygon: 

  1' 2' 3'
x 50 30 50
y 40 40 20
