STR-21 - Reticular Beam

For an isostatic truss, externally constrained with a hinge and a support, and loaded into the nodes, the program calculates the reactions of the constraints and the stresses in the structural rods.

The only limitation to the geometry of the truss is that each node (n) must be connected to the nodes (n+1) and (n+2). This implies that no more than 4 rods can converge in each node.

To provide the geometric data of the structure it is necessary to set a reference system that originates in the hinge joint with the Y axis coinciding with the vertical reaction. The verses can be set at the operator's discretion.

This type of numbering and this reference system is easily applicable to most of the trusses of practical use as shown in the following figures.

In some cases, with an artifice of calculation, it is possible to calculate a truss that has one or more nodes with 5 converging rods. For example, the truss shown below can be calculated by transforming it as shown in the next figure 5.

Assigning a very small length (2 cm) to the fictitious beam 4-6 and dividing the load present in the real node 4 into the fictituous nodes 4 and 6, the stress in the real rod 4-5 will be equal to the sum of the stress in the fictitious rods 4-5 and 5-6.

For each node (n) it is necessary to provide:

  • Xn = abscissa;
  • Yn = ordinate;
  • PXn = nodal load in the direction of the X axis;
  • PYn = nodal load in the direction of the Y axis.
  • The loads are positive if the direction coincides with that of the axes.
  • The constraint reactions of the hinge, RCX and RCY, and of the support, RAY, are calculated with the equations of translation and rotation equilibrium of the Science of the Constructions.

The units of measurement must be: lengths in meters, forces in Newton and related multiples or submultiples, with the following limitations:

|Xn|<5000; |Yn|<5000; |PXn|<50000; |PYn|<50000.

Component X and Y of the total stress Fn present in node "n":

|FXn|<50000; |FYn|<50000.

Total number of nodes: N≤45.

Setting higher values the program stops flashing the wrong value.

Description Of The Method And Formulas Used

If we call FX and FY the sum of the external forces acting in the node "n" (external load and support reactions) in the X and Y directions, and if we call ΣNX and ΣNY the sum of the 4 internal converging rods, in the X and Y directions, for the equilibrium of the node must be:

FXn + NX1 + NX2 + NX3 + NX4 = 0

FYn + NY1 + NY2 + NY3 + NY4 = 0

Then the system of 2N equations in 4N unknowns (N = total number of nodes) cannot be solved.

For the particular type of numbering adopted, and for the particular chosen of the reference system, the node of the support or that of the hinge, will only have two rods. By assigning the number n=1 to this node, the related equation system will be:

PX1 + NX12 + NX13 = 0 (X-component)

PY1 + NY12 + NY13 = 0 (Y-component)

PX1 and PY1 are the RX and RY components of the constrained reaction R in the node 1.

NX12 NX13 NY12 NY13 are the X and Y component of forces N12 and N13 in the rod 1-2 and 1-3

As

NX12 = N12٠(X2 - X1) / L12

NY12 = N12٠(Y2 - Y1) / L12

NX13 = (X3-X1) / L13

NY13 = N13٠(Y3 - Y1) / L13

in which

L12 = √[(X2 - X1)2 + (Y2 - Y1)2] is the lenght of the rod 1-2

L13 = √[(X3 - X1)2 + (Y3 - Y1)2] is the lenght of the rod 1-3

Placing :

X12 = (X2 - X1) / L12 and Y12 = (Y2 - Y1) / L12

X13 = (X3 - X1) / L13 and Y13 = (Y3 - Y1) / L13

and by substituting these in the two equilibrium equations, we obtain

RX1 + N12٠X12 + N13٠X13 = 0

RY1 + N12٠Y12 + N13٠Y13 = 0

The system of two equations in two unknowns applied in node 1, will therefore be resolvable by replacement and will provide for the rods 1-2 and 1-3, the following solutions:

N12 = (RY1٠X13 - RX1٠Y13) / (X13٠Y12 - X12٠Y13)

N13 = (RX1٠Y12 - RY1٠X12) / (X13٠Y12 - X12٠Y13)

In the next node n.2 three rods will converge: rod 1-2, already calculated, and rods 2-3 and 2-4 still to be calculated.

Writing the two equations for the equilibrium of this node we will obtain the system of two equations in two unknowns:

FX2 + N23٠X23 + N24٠X24 = 0

FY2 + N23٠Y23 + N24٠Y24 = 0

The system will be solved by the same method.

In the next node n.3 will converge: rod 1-3 and rod 2-3, already calculated, and rods 3-4 and 3-5 still to be calculated.

Having only two unknowns, the forces N34 and N35 can be calculated in a similar way to the previous node.

For the subsequent nodes the unknown forces acting in the rods converging in them will always be two and can be calculated in a similar way.

Instructions

The instructions for using the program are summarized in the "card emulation” similar to the magnetic card of the TI-59.

It summarizes the instructions assigned to each definable key, transcribed in the box referring to the key. Each input procedure is independent and the operator can perform it without a predetermined order.

  1. To start the program

    The program calculates the forces in the rods for a reticular beam with a maximum of N = 45 nodes.

    enter N and press the A key. If you enter N > 45 the program stops flashing 45. In this case repeat the procedure.

    To avoid the total entry of new data, many of which would be repetitive between similar structures, the program deletes only the registers containing the forces calculated in the nodes of the previous structure and limited to the number of nodes N of the new structure.

    If the new data is very different from the previous one, it is convenient, before starting the program, press 2nd Cms to clear all registers.

    With the procedure assigned to key B, C and D it is possible, at any time, to recall the data of nodes and of the constraints of the structure.

  2. Enter the coordinates of node "n"
    • Enter "n" and press B key.
    • The calculator display the stored data of Xn. If it is incorrect enter the new value and press R/S else press only R/S.
    • The calculator display the stored data of Yn. If it is incorrect enter the new value and press R/S else press only R/S.
    • The calculator display 0 to indicate that the procedure has been completed.

  3. Enter the load in node "n"
    • Enter "n" and press C key.
    • The calculator display the stored data of PXn. If it is incorrect enter the new value and press R/S else press only R/S.
    • The calculator display the stored data of PYn. If it is incorrect enter the new value and press R/S else press only R/S.
    • The calculator display 0 to indicate that the procedure has been completed.

  4. Enter the number of nodes of the hinge and the support
    • Press D key.
    • The calculator display the stored data of NC (number of node of the hinge). If it is incorrect enter the new value and press R/S else press only R/S.
    • The calculator display the stored data of NA (number of node of the support). If it is incorrect enter the new value and press R/S else press only R/S.
    • The calculator display 0 to indicate that the procedure has been completed.
    • If enter NC=0 and NA=0 the program will not calculate the support reactions which, however, must be calculated by the operator and stored as external forces acting in the relative nodes in which the structure is constrained. In this case, of course, the configuration of the supports has no limitations.

  5. Calculation and printing of results
    • Pressing E, the program print data and results for all nodes.

      At the end the calculator display 0 to indicate that the procedure has been completed.

      Pressing R/S the program will display the forces in the rods, starting from node "1", displaying the number of the rod 1-3 in the form 1.03. Pressing R/S in sequence will display the force N and its components NX and NY.

      Pressing R/S again in sequence will display the same results for the rod 1.02.

      Finally, pressing R/S again will display the forces in the rods of the n=2, starting first with the rod 2.04 and then the rod 2.03.

      After displaying the forces in the rods of the last node, pressing R/S again repeats the cycle starting from the rods of node 1.

      Press CLR before starting another procedure.


The instructions for using the program will be illustrated directly with an example.

    Number of steps: from 000 to 929.

    Flags used: 1 , 2 , 7
