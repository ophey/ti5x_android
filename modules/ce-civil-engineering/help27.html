STR-27 - BEAM STRESSES

In a beam, or beam segment, of length L, the program calculates the bending moment M and the shear T in any section chosen by the operator, of coordinate "X" with respect to the extreme left.

Different load conditions are possible, combinable in any order:

  1. uniform load "Q" along the whole beam;
  2. two bending moments, MA and MB, applied at the ends;
  3. concentrated force F in the coordinate point XF;
  4. distributed load linearly variable from Q1 to Q2, applied from the coordinate X1 to X2.

The units of measurement must be consistents and those of the International System: lengths in meters; loads in Newton and related multiples or submultiples.

The sign of the bending moment and shear stress must respect the "phisical convention", Positive moment if right-handed; positive shear if directed upwards in the left section, and downwards in the right section.

The program uses only the memory registers from 90 to 99. Therefore it can be recalled from the library also during the use of the structural calculation programs from STR22 to STR26 without altering the contents of their archives.

Method Of Calculation And Formulas Used

With reference to Figure 1, called

L = length of the beam;

Q = uniform load intensity on the beam;

F = intensity of the concentrated force F applied in the XF coordinate;

MA, MB = intensity of the bending moments each acting in the extreme nodes A and B of the beam;

Q1, Q2 = initial (left) and final (right) values of the intensity of the linearly variable load applied on the beam from the X1 to X2 coordinate.

P = total load on the beam;

X = coordinate, measured from the left node A, of the section of the beam in which to calculate:

MX, TX = bending moment and shear in the "X" section produced by the single load;

∑MX, ∑TX = total bending moment and shear in section "X".

The program calculates the bending moment MX and the shear TX in the X coordinate produced by each type of load using the following formulas

  1. Uniform load Q across the beam:

    Total load: P=QL;

    Shear: TX= Q(L/2-X);

    Bending moment: MX= -(QX/2)(L-X)

  2. Bending moments MA and MB in the nodes:

    Total load: P=0;

    Shear: TX=-(MA+MB)/L;

    Bending moment: MX=-MA -TX x (X).

  3. Concentrated force F in the XF coordinate:

    Total load: P=F;

    if XF≥X:TX= F-FxXF/L;  MX= -TX x (X);

    if XF<X:  TX= -FxXF/L;  MX= -TX x (L-X);

  4. Linearly distributed load from Q1 to Q2 in coordinates from X1 to X2:

    Total load: P = (Q1+Q2)(X2-X1)/2

    Center of gravity coordinate of P: XG = X1 + [(Q1+2Q2)/(Q1+Q2)]x[(X2-X1)/3]

    Vertical reaction in node A to the left: RA= P - P x XG/L;

    Vertical reaction in node B to the rigth: RB= P x XG/L;

    If X≤X1:  TX= P-RB;  MX=-TX x (X);

    If X<X2:  TX= -RB;MX=-TX x (L-X);

    If X1<X≤X2:

    TX= RA - Q1(X-X1) - [(Q2-Q1)/2] x (X-X1)²/(X2-X1);

    MX= -RAxX + Q1(X-X1)²/2 + [(Q2-Q1)/6] x (X-X1)³/(X2-X1);

At the end the program calculates the bending moment ∑MX and the shear ∑TX in the same coordinate as the sum of the relative stresses MX and TX.

The physical convention of signs is used.

Instructions For Data Input And Printing Results

The instructions for using the program are summarized in the "card emulation”, similar to the magnetic card of the TI-59, which appears below the display when the APP calls the program from the library module. It summarizes the instructions assigned to each definable key, transcribed in the box referring to the key.

For data input the program allows the following procedures which can be started without a specific order:

RESET DATA-ARCHIVE

Enter "L" and press SBR CLR.

The program reset the values of loads and their coordinates stored for a previous calculation.

At the end the display show "0".

SET THE UNIFORM LOAD "Q" AND LENGTH "L":

Press A key. Diplay = stored value of "Q".

If it is "0" or a value incorrect, enter the new value.

Then press R/S to display "L".

If it is incorrect, enter the new value and press R/S.

At the end display shows "0" to indicate the end of the procedure.

SET THE BENDING MOMENTS "MA" AND "MB" AT HE ENDS:

Press B key. Diplay shows the stored value of "MA".

If it is incorrect, enter the new value.

Then press R/S to display "MB".

If it is incorrect, enter the new value and press R/S.

At the end display shows "0" to indicate the end of the procedure.

SET THE CONCENTRATED FORCE "F" AND ITS "XF" COORDINATE:

Press C key. Diplay shows the stored value of "F".

If it is incorrect, enter the new value.

Then press R/S to display "XF".

If it is incorrect, enter the new value and press R/S.

At the end display shows "0" to indicate the end of the procedure.

SET LINEAR DISTRIBUTED LOAD FROM "Q1" TO "Q2" AND ITS COORDINATES:

Press D key.

Diplay shows the stored value of "Q1". If it is incorrect, enter the new value.

Press R/S to display "X1".

If it is incorrect, enter the new value and press R/S.

Diplay shows the stored value of "Q2". If it is incorrect, enter the new value.

Press R/S to display "X2".

If it is incorrect, enter the new value and press R/S.

At the end display shows "0" to indicate the end of the procedure.

SET "X" COORDINATE IN WHICH TO CALCULATE "MX" AND "TX" STRESSES.

Enter the value of "X" and press E key.

The program calculate the bending moment "MX" and the shear "TX" in the section "X" distant from the left end of the beam.

At the end display show "MX". Press X⇌T key to show "TX".

In addition the program prints a report in the phone video thet includes the geometric data of the beam, those of the loads present with the relative application coordinates and the value of "MX" and "TX" stresses calculated in section "X".

Warning

If the new beam has loads and/or dimensions different from the previous one, the operator can only modify the data that change otherwise, he must first reset all the data with the procedure "1".

For loads Q, F and M, the program accepts integer, positive or negative, less than "500'000" in absolute value. The length "L" of the beam and the coordinates of the loads must be integer, positive and less than "10'000".

The program eliminates any decimal digit from the load value, and the decimal digits beyond the second from the coordinates value.

The value of the length "L" and "X" coordinate in which to calculate the M and T stresses, must be positive. In any case, the program only stores its absolute value.

The "X" coordinate must be less than or equal to "L".

Otherwise the program stops flashing the limit value not allowed. In this case, press CLR to eliminate the blinking and repeat the procedure.

To view the print results report, touch the printer icon at the top center of the app's main screen.

The results printed in the report are accurate.

The shear TX due to the load Q, or F, or (MA+MB) is equal and opposite to the relative constraint reaction (450 N, 40 N and 32 N) on the left node of the beam .

The ∑TX shear in the section X=0 is positive (directed downwards), equal to the sum of the shear due to the single load, and is equal and opposite to the constraint reaction in the left node of the beam.

The moment ∑MX to the right of the same section is equal and opposite to the bending moment MA applied in the left node.
