STR-04 - Continuous Beam

The program calculates the moments on the supports of a continuous beam with a maximum of 16 supports (increased to 18 using the “Ti5x app”). The input data are the moments of perfect embedding and the stiffness of each span. The moments at the ends of each span must first be calculated with the STR−03 program. It is possible to treat beams with joints or jumps at the ends. If the extreme support is a perfect joint, replace it with a span of infinite stiffness and zero moments. The section of each span must be constant.

Description Of The Method Used

The program uses Kani's method. For setting the data and presenting the results, the supports are numbered from left to right. Moments of perfect fit are assumed to be negative (according to the construction science convention) and not greater than 99999. The sign can be omitted. The scores of the results follow the construction science convention. The precision set by the program is 1 on the rates.

M3,4 = 2000 Kg/m
M2,3 = M3,2 = 4166 Kg/m
M1,2 = M2,1 = 666 Kg/m
M1,0 = 0 Kg/m
K1,2 = 0.5
K2,3 = 0.2
K3,4 = 0
K1,0 = ∞

To calculate the continuous beam it is necessary to first calculate, for each span and for any shelf, the moments at the ends using the STR03 program considering each span perfectly fitted at the two ends and the shelf perfectly fitted to one end.

To calculate the beam in the figure, replace the joint with a span of stiffness K = "9999999999" equal to the maximum value that can be set with this program. The units of measurement of the forces and moments indicated in the figure must be replaced with those of the International System, 1 kg = 10 N (Newton) = 1 dN (decaNewton), 1 kgm = 1 dNm = 1 KNcm. The program solves the system of equations by continuous iteration.

Example

  1. Print option: 2nd St flg 1
  2. Initialization, press E
  3. Set number of supports, enter 4, press 2nd E', display 3.
    Delete the T register: CLR X⇌T
  4. Set data of support "3":
    • Perfect joint moment MIi:
      • to the right, enter 2000, press A, display 2000.
      • to the left, enter 666, press R/S, display 0.00666.
    • Stiffness K in the pan:
      • to the right, enter 0, press R/S, display 0.
      • to the left, enter 0.2, press R/S, display 2.
  5. Set data of support "2":
    • Perfect joint moment MIi:
      • to the right, enter 4166, press A, display 4166.
      • to the left, enter 666, press R/S, display 0.00666.
    • Stiffness K:
      • to the right, enter 0.2, press R/S, display 0.2.
      • to the left, enter 0.2 (incorrect data), press R/S, display 1.
  6. Set data of support "1":
    • Perfect joint moment MIi:
      • to the right, enter 666, press A, display 666.
      • to the left, enter 0, press R/S, display 0.
    • Stiffness K:
      • to the right, enter 0.5, press R/S, display 0.5.
      • to the left, enter 9999999999 (infinite stiffness), press R/S, display 0.49999.
  7. Data correction procedure.
      Set data of support in which there is incorrect data:
    • to the right, enter 2, 2nd A', display 2.
    • Set data of support "2":
      • Perfect joint moment MIi:
        • to the right, enter 4166, press A, display 4166.
        • to the left, enter 666, press R/S, display 0.00666.
      • Stiffness K:
        • to the right, enter 0.2, press R/S, display 0.2.
        • to the left, enter 0.5 (correct data), press R/S, display 1.
    • Start the calculation, press B.

If you enter MIi > 99999, the display flashes. In this case, press CLR and repeat the data input of the support “n”.
