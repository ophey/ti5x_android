STR29 - STIFFNESSES 2

The program calculates the stiffness "K" of a beam as the ratio between the moment of inertia "J" of the section with respect to its barycentric axis, and the length "L" of the beam (K=J/L).

The stiffness is calculated for the following sections:

The section is considered constant for the whole length.

Any unit of measurement can be used.

The program uses the registers from 90 to 99 and therefore can be recalled during the elaboration of the structural programs from STR22 to STR26 without interfering with their archives. It will be sufficient to transfer the value of the stiffness from the display to the structural program or others applications with the "Copy/Paste" function of the "Ti5x" App, illstrated in the instruction manual.

Instructions

The instructions for using the program are summarized in the "card emulation” similar to the magnetic card of the TI-59, which appears below the display when the APP calls the program from the library module.

It summarizes the instructions assigned to each definable key, transcribed in the box referring to the key.

The “>“ symbol indicates to start the procedure by pressing its definable key to display the first data.

The symbol ":" after the data indicates to press the R/S key to store it and display the next data or complete the procedure.

  1. AKey: Provide the length L.

    Press A: display = stored value of "L".

    If it is incorrect enter new value and press R/S.

    Display = "0" (procedure completed).

  2. BKey: provide the "B", "B0" and "B1" bases of the section.

    Press B: display = "B" (stored value).

    Enter the new value (only if different) and press R/S to store it.

    Display = "B0". Enter the new value (only if different) and press R/Sto store it.

    Display = "B1". Enter the new value (if different) and press R/Sto store it.

    Display = "0". End of the procedure.

  3. CKey: provide the "H", "H0" and "H1" heights of the section.

    Press C: display = "H" (stored value).

    If different, enter the new value and press R/Sto store it and display "H0".

    If different, enter the new value and press R/Sto store it and display "H1".

    If different, enter the new value and press R/Sto store it.

    Display = "0". End of the procedure.

  4. Select the section of the beam and calculate the stiffness "K":

    Press 2ndA'key to select "double T-section";

    or 2ndB'for "C-section"

    or 2ndC'for "H-section"

    or 2ndD'for "rev.T-section".

    The program display the stiffness Kof the beam.

    Press X=Tto displays "A" (section area).

    The program also print a report whit data, and results "X" (neutral axis), "A" (area of the section), "J" (moment of inertia) and "K" (beam stiffness).

    If the lengths are measured in cm, area "A" will be expressed in cm2, moment of inertia "J" in cm4and stiffness "K" in cm3.

    To view the report press the printer icon at the top center of the app's main screen.

  5. For a new calculation, reset only the data that changes.

Formulas Used

With reference to previous figure, called

L = length of the beam;

A = section area;

S = section static moment;

J = moment of inertia;

X = neutral axis = S/A;

K = stiffness of the section = J/L;

the program calculates the stiffness using the following formulas:

  1. double T-section:

      A = B0H + (B - B0)H0 + (B1 - B0)H1

      S = [B0H² + (B - B0)H0² + (B1 - B0)H1 (2H - H1)] / 2;

      X = S / A;

      J = [BX3 - (B - B0)(X - H0)3 + B1(H - X)3 - (B1 - B0)(H - X - H1)3] / 3;

  2. C-section:

    the formulas coincide with the double T-section

  3. H-section:

    A = B0H + B1H1 + (B - B0 - B1)H0;

    X = H / 2;

    J = [B0H3 + B1H13 + (B - B0 - B1)H03] / 12;

  4. RT-section:

    A = B0H + (B - B0)H0;

    X = H / 2;

    J = [B0H3 + (B-B0)H03 ] / 12;

Example 1

With reference to the previous figure, calculates the stiffness of a I-section beam having the following dimensions:

L= 500 cm; B=50 cm; Bₒ=20 cm; B₁=70 cm; H=100 cm; Hₒ=10 cm; H₁=10 cm.

The stored data is assumed to be zero.

  1. Provide the data:
    • Set L:

      press A. Display = 0(stored value of L).

      Enter 500and press R/S.

      Display = 0. End of procedure.

    • set B, Bₒ and B₁.

      press B. Display = 0(stored value of B).

      It is incorrect. Enter 50and press R/S.

      Display = 0(stored value of B0).

      It is incorrect, then enter 20and press R/Sto store it.

      Display = 0(stored value of B1).

      It is incorrect, then enter 70and press R/Sto store it.

      Display = 0. End of the procedure.

    • set H, H0, and H1.

      press C. Display = 0(stored value of H).

      It is incorrect. Enter 100and press R/S.

      Display = 0(stored value of H0).

      It is incorrect. Enter 10and press R/Sto store it.

      Display = 0(stored value of H1).

      It is incorrect. Enter 10and press R/Sto store it.

      Display = 0. End of procedure.

  2. Results:

    Press 2ndA'.

    The program display K = 6528,809524cm3.

    Press X=Tto displays area A = 2800cm2.

Example 2

Compared to the previous example, only the values of B0, H and H0change.

      Display = 70(stored value of B1).

      press C. Display = 100(stored value of H).

      It is incorrect. Enter 50and press R/S.

      Display = 10(stored value of H0).

      It is correct.

      In RT-section there is no element of width H1and the program does not use the value stored in the archive. So we can stop the procedure here. Press CLR.

  2. Results:

    Press 2ndD'.

    The program display K= 215.cm3.

    Press X=Tto displays area of section: A = 900cm2.

NEWS ABOUT THE PROGRAM

Number of steps: from 000 to 477.

Registers used: from 90to 99.

Flags used: 0, 1.

Labels: 2nd NOP, - , Yx, × , EE, RCL, Int, STO, Rad, Grad, sin, D.MS, SUM and definable keys A, B, C, D, A', B', C', D'
