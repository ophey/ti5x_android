STR-03 - Single Span Beams

The program calculates shear and bending moment in equidistant points whose number is set by the operator. The loads, applicable separately or in combination, can be: moment at the extremity, force concentrated in any point, uniform load.

There are four constraint conditions:

  • Left Joint:
  • Support - Support
  • Joint - Support
  • Joint - Joint
  1. Print option: 2nd St flg 1
  2. Provide data:
    • Beam length: enter L, press A
    • Concentrated load: enter P, press B
    • Load abscissa: enter E, press R/S
    • Uniform load: enter Q, press C
    • Moment to the left: enter Ms, press D
    • Moment to right: enter MD, press E
  3. Set the number of subdivisions of the beam, enter n, 2nd A'
  4. Select the constraint conditions and start calculation:
    • Left joint, 2nd B'
    • 2 supports, 2nd C'
    • Left joint and right support, 2nd D'
    • 2 joints, 2nd E'

    Results in "n+1" coordinates, from X=0 to L:

    • Xn is displayed,
    • press R/S M(Xn) is displayed: bending moment in Xn
    • press R/S, T(Xn) is displayed: shear moment in Xn
    • press R/S, continue in the next coordinate n+1

    If flag 1 has been activated, the results will be printed in automatic sequence.

  5. For a new calculation go back to step 2.

    To calculate the beam, the program uses only the necessary data according to the assigned constraints. For another type of beam, there is no need to reset the unchanged data.

Example

Calculate a beam of the Joint-Support (fig.3): 

    L=300 cm    P=500 dN    E=150 cm
    Q=2 dN/cm   MD=0 dN/cm
  1. Print option: 2nd St flg 1
  2. Provide data:
    • Beam length: enter 300, press A
    • Concentrated load: enter 500, press B
    • Load abscissa: enter 150, press R/S
    • Uniform load: enter 2, press C
    • Moment to right: enter 0, press E
  3. Set the number of subdivisions of the beam, enter 10, 2nd A'
  4. Select the constraint conditions and start calculation:
    • Left joint and right support, 2nd D'
