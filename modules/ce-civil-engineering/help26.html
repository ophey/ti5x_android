STR-26 - FIXED NODE STRUCTURE

General Description Of The Program

The program calculates the bending moment at the ends of the rods of a structure having any geometry.

Fig.1 illustrates a scheme of a structure that can be calculated with this program:

If N= number of nodes and R= number of rods of the structure, their sum must be less than or equal to 80.

The number of rods converging in a node is less than or equal to 4.

The rods are connected to the X and Y nodes, X to the left of the horizontal rods or to the base of the vertical ones, Y to the right or top. The section of each rod is considered constant.

The horizontal components of the forces are opposed by the external constraints of the structure.

The units of measurement must be consistents and those of the International System: lengths in meters, moments in kNm.

The loads are assumed positive if directed downwards and to the right.

The data to be provided to the program are the stiffness K of the rods and the moments of perfect interlocking MIX and MIY at their ends.

The sign of MIX and MIY the perfect interlocking moment is positive if right-handed. If the load sign is positive, they will be negative to the left of the horizontal rods and at the base of the vertical rods.

The value of the perfect interlocking moment must be integer and included between "-499" and "+499".

The stiffnesses will be set as the ratio between the moment of inertia "J" of the rod section (constant) and its length "L", and therefore expressed in cm3. Their value must be integer and included between "0" and "999". The program considers the maximum value "999" as "infinite stiffness" and during the calculation it amplifies it adequately.

The stiffnesses "K" and moments "MIX" and "MIY" can also be calculated with the "STR-28", "STR-29", and "STR-30"" programs of the "Civil Engineering" library module.

Calculation Method And Print Results

The program calculates the rotation "R=2Ef" of the nodes with the Gehler-Takabeya method, where "f" is the rotation measured in radians and "E" is the elastic modulus of the structure, measured in N/cm2. The sign of R is positive if right-handed.

The system of N equations is solved with Gauss-Seidel iterative method and the iterations continue until the increment of each unknown is less than the limit "ԑ" imposed by the operator (Belluzzi - Construction Science, Vol.II/p.465, Zanichelli Ed. - Italy).

After having solved the system of equations (time about 5 second / node), the program, prints, for each rod, the geometric and elastic data, the rotations "RX" and "RY" of the extreme nodes, and the bending moments "MX" and "MY" at the ends.

By pressing the SBR key twice the program will print the whole report.

It is also possible to print the results for a single rod chosen by the operator.

Example 1 will illustrate how to calculate the structure of Fig.1.

Procedures For Data Input And Printing Results

The instructions for using the program are summarized in the "card emulation" similar to the magnetic card of the TI-59, which appears below the display when the APP calls the program from the library module.

To start the program there is a specific procedure assigned to the 2nd A' key, which resets the registers in which to store data and results of the rods and nodes.

There are also 4 distinct data input procedures, respectively assigned to the A key, B, C and D keys of the calculator.

Except for the procedure [A'], the other procedures can be started without a specific order.

If the new structure has different dimensions from the previous one, the operator must start procedure [A'] to initialize the data archive, otherwise it will use procedures [A], [B], [C] and [D] to modify only the data that changes.

The instructions for using the program are supplied directly with the following example 1.

[C] key - Set The Stiffness Of The Rod "r"

The program accepts values of "K" between 0 and 999 without decimal digits. It considers the value "999" as an infinite stiffness and during the calculation amplifies it adequately.

If some or all of the values of K are greater than 998, the structure can also be calculated by proportionally multiplying them by the scale factor S<998/KM, where KM is the maximum value between the stiffnesses present in the structure.

In example 1, the section of the rod 4 is 30x50 cm, J=30x503/12 and the stiffness is KM=J/L= 312500/300 = 1041 cm3, greater than 998. To calculate the structure, all stiffnesses must be stored multiplied by a scale factor SK<998/KM =0.959. In example 1 they have been multiplied by SK=0.95 and the relative values are shown in fig.3 and in table 2.

At the end of the calculation, the actual rotation of the nodes will be equal to RxSK.

[E] key - Structure Calculation And Print The Results

If we have already memorized the "IDR" data (procedure "A"), the "IDN" data (procedure "B"), the stiffnesses "K" (procedure "C"), the "MIX" and "MIY" moments in the rods (procedure "D"), now we can calculate the structure.

Enter the calculation tolerance "ε" and press E.

The tolerance is equal to the absolute value of the difference betwen to the value of "R" rotations, calculated in two successive iterations: "ε"=|R'-R"|. It is recommended to use values of "ε" between "0" (maximum precision) and "9".

The program calculate "R" for all the nodes in according to Gehler-Takabeya method, and considering the fixed nodes and the inextensible rods.

Solve the system of equation with the iterative method of Gauss-Seiler, printing a "0" at the start of each new iteration.

The iterations will end when the tolerance is respected in all nodes.

Finally print a report with the rotations "R", and the bending moment MX and MY in all rods.

At the end the display flashes the number of rod "R"+1. Press CLR to stop blinking and clear the display.

Warning:

During the calculation the absolute values of the rotation of the node must be less than 100000, otherwise the program stops flashing the calculated value RX. In this case it is necessary to press CE to stop the flashing, note the RX value, and repeat the input of the stiffnesses (procedure C) with values proportionally increased by multiplying them by an appropriate scale "SR" greater to RX/100000, and recalculate the structure with procedure E.

Check Of The Results

The accuracy of the results obtained with this program can be verified by adding the moments present in the rods around each node. For the equilibrium of the structure their sum should be zero.

From figure 4 of the example 1, we verify:

Node 1: +104 -113 +0.4 +8.43 = -0.17; % error = 0.17/113 = 0.15%

Node 2: +23.3 -23.3 = 0; % error = 0

Node 3: +347 +19.7 +41.7 -409 = -0.6; %error = 0.6/347 = 0.17%

Node 4: +249 -249 = 0; %error = 0

Node 5: +337 -216 -122 = -1; % error = 1/337 = 0.3%

The maximum percentage error is in node 5, equal to 0.3% with respect to the value of the maximum moment in the node.

From figure 6 of example 2, we can verify that the results are exactly symmetrical, that the rotation in node 2 is zero and that the sum of the moments in the nodes is also zero.

The results obtained can therefore be considered exact in relation to the iterative calculation adopted for the solution of the system of equilibrium equations.
