STR-07 — Beam on Elastic Ground

The program calculates, according to winkler's hypothesis, the sag, contact stress, moment and shear at a point of a constant section beam on elastic ground subjected to N vertical loads (N ≤40).

L = the beam length (cm)
a = abscissa of the load (cm)
B = support beam width (cm)
y = elastic sag at point “x” (cm)
K = constant background (dN/cm3)
σt = stress in the ground at point “x” (dN/cm2)
J = moment of inertia section (cm4)
M = bending moment at point “x” (dNcm)
E = elasticity modulus (dN/cm2)
T = shear stress at point “x” (dN)
P = load intensity (dN)

the program calculates, at the point "x" chosen by the operator, the values of y(x), σt(x), M(x), T(x), as calculates the values as the sum of the values generated by each single load “Pi” (dN) < 106 applied in point ”ai” (cm) < 104. If you set “N” > 40 or “P” ≥ 106 or “a” ≥ 104, the program gives incorrect results.

Example

P1 = 10000 dN
P2 = 20000 dN
P3 = 12000 dN
B = 100 m
K = 20 Kg/cm2
J = 1000000 cm4
E = 200000 Kg/cm2
  1. Print option: 2nd St flg 1
  2. Set data beam and ground:
    • length L, enter 1600 cm, press A, display 1600.
    • base width B, enter 100 cm, press R/S, display 100.
    • constant background K, enter 20 kg/cm2, press R/S, display 2000 (*).
    • mom. inertia J, enter 1000000 cm4, press R/S, display 0.002 (*).
    • elasticity modulus E, enter 200000 kg/cm2, press R/S, display 11.3137085 (*).
    • number of loads N ≥ 40 (*), enter 3, press R/S, display 1.
  3. Set data loads (P < 106, a < 104) (*):
    • data load 1: P1, enter 10000 dN, press B, display 10000.
      a1, enter 300 cm, press R/S, display 2.
    • data load 2: P2, enter 20000 dN, press B, display 20000.
      a2, enter 700 cm, press R/S, display 3.
    • data load 3: P3, enter 12000 dN, press B, display 12000.
      a3, enter 1300 cm, press R/S, display 0.
    • (repeat from 1 to N loads)
  4. Results in the X1 coord, enter 900 cm, press C: 
               900.     X
    0.008641235     Y
    0.172824693     Σ
   -172956.7704     M
   -693.4054031     T

    Results in the X2 coord, enter 1300 cm, press C: 
              1200.     X
     0.01383298     Y
    0.276659597     Σ
    10893.36357     M
    2553.937606     T
  5. For a new calculation, go back to step 3 if only the load data changes, else go back to step 2.

(*) NOTES: If you set “N” > 40 or “P” > 999999 or “a” > 9999, the program gives incorrect results.
