STR-06 - Wedged Portal

The program calculates the support reactions, the moments at the nodes and the moments in the position of the load in a symmetrical wedged portal. Three load conditions are foreseen: uniform load Q on the beam, concentrated load PV in any position A2 on the beam, concentrated load PR in any position A1 on the pillar. The section J of pillars and beam must be constant.

K = J2 ⁄ J1
D1 = K + 2
D2 = 6 K + 1
B1 = H - A1
B2 = L - A2
α1 = A1 ⁄ H
α2 = A2 ⁄ L
β1 = 1 - α1
β2 = 1 - α2

Example

  1. Print option: 2nd St flg 1
  2. Set geometric data:
    • height, enter 300 cm, press A, display 300.
    • length, enter 500 cm, press B, display 500.
    • mom. inertia J1:, enter 160000 cm4, press C, display 160000.
    • mom. inertia J2:, enter 312500 cm4, press D, display 312500.
  3. Set the load and start the calculation:
