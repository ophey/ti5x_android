STR18 − Rectangular Section in Reinforced Concrete - Shear Reinforcement

The program calculates the shear reinforcement of a beam section reinforced concrete rectangular section.

The condition imposed by the operator is the area of the bend irons present in section.

The units of measurement must be consistents and those of the International System: lengths in meters, forces in Newton and related multiples or submultiples.

The sign of the shear stress in the section is positive according to the construction science convention.

Symbols

  • L = Length of beam section (cm)
  • H = useful section height (cm)
  • B = beam section width (cm)
  • TMX, TMIN = max. and min. shear stress in the section (with equal sign)(dN)
  • TAU = maximum tangential stress in the section (dN/cm²)
  • N = number bracket arms
  • IP = inclination of bent bars with respect to beam axis (degrees)
  • AP = = total area of ​​the bent bars in the section (cm²)
  • DS = iron diameter of the brackets (cm)
  • PS = brackets step (cm)
  • ASM = total area of ​​the brackets per meter (cm²)
  • ΩT = area of ​​the shear diagram in the section (dN)
  • ΩS = part of the ΩT absorbed by the brackets area (dN)
  • ΩP = part of the ΩT absorbed by the bent bars area (dN)
  • %S = percentage of the area of ​​the shear diagram absorbed by the brackets
  • ΣF = stress in irons (dN/cm²)
  • RCK = concrete characteristic resistance (dN/cm²)

The italian legislation requires:

  • ASM ≥ 3 cm²
  • PS ≤ MIN(0.8H, 33cm);
  • %S ≥ 40 %.

Instructions

The instructions for using the program are summarized in the "card emulation” similar to the magnetic card of the TI-59.

It summarizes the instructions assigned to each definable key, transcribed in the box referring to the key.

Each input procedure is independent and the operator can perform it without a predetermined order.

The “>“ symbol indicates to start the procedure by pressing its definable key to display the first data.

The symbol ":" after the data indicates to press the R/S key to store it and complete the procedure or display the next data.

At the end, the procedure displays "0".

  1. Provide the data:
    Each input procedure is independent.
    • Set L (cm):
      press A. Display = stored value of L.
      If it is incorrect enter the new value and press R/S.
      Display = 0. End of procedure.

    • set H, B (cm):
      press B. Display = stored value of H (cm).
      If it is incorrect enter the new value.
      press R/S. Display = stored value of B (cm).
      If it is incorrect enter the new value and press R/S to store it.
      Display = 0. End of procedure.

    • set TMX, TMIN (dN):
      press C. Display = stored value of TMX (dN).
      If it is incorrect enter the new value.
      press R/S. Display = stored value of TMIN (dN).
      If it is incorrect enter the new value and press R/S to store it.
      Display = 0. End of procedure.

    • set ΣF, RCK (dN/cm²):
      press D. Display = stored value of ΣF (dN/cm²).
      If it is incorrect enter the new value.
      press R/S. Display = stored value of RCK (dN/cm²).
      If it is incorrect enter the new value and press R/S to store it.
      Display = 0. End of procedure.

    • set N, IP:
      press E. Display = stored value of N.
      If it is incorrect enter the new value.
      press R/S. Display = stored value of IP (degrees).
      If it is incorrect enter the new value and press R/S to store it.
      Display = 0. End of procedure.

    • set AP (cm²):
      press 2nd A'. Display = stored value of AP (cm²).
      If it is incorrect enter the new value and press R/S.
      Display = 0. End of procedure.

    • set DS (cm):
      press 2nd B'. Display = stored value of DS (cm).
      If it is incorrect enter the new value and press R/S.
      Display = 0. End of procedure.

  2. Results:

    Pressing 2nd E', the program calculates the tangential stress TAU in the concrete and compares it with the maximum value T1 and the minimum T0 allowed by Italian law for the RCK class of concrete.

    If TAU > T1 the program stops and flashes TAU to advise the operator to modify the resistant section of the beam.

    If TAU < T0 the program assumes ASM = 3 cm²/meter and %S= 100%.

    If T0 < TAU < T1 the program calculates ΩT, ΩP, ΩS and %S.

    If %S < 40 the program sets ΩS= 0.4 ΩT; %S = 40% and calculates ASM.

    If ASM < 3 cm²/meter the program sets ASM = 3 cm²/meter and %S = 100%. Then calculate the step of brackets PS (cm).

    If PS ≥ Min(0.8H, 33cm) the program sets PS = MIN(0.8H, 33cm) and flashes PS to advise the operator to choose a smaller DS diameter.

    At the end the program displays %S. Pressing R/S in sequence will display the other results ASM and PS.

    The programm will also print a report with the data and results.

    For a new calculation, reset only the data that changes.

Example 1

Calculate the shear reinforcement of a section of beam having the following characteristics:

  • L=200 cm
  • H=47 cm
  • B=30 cm
  • TMX=15000 dN
  • TMIN=8000 dN
  • ΣF=1900 dN/cm²
  • RCK=250 dN/cm²
  • N=2
  • IP=45°
  • AP =6,03 cm²
  • DS=1 cm.
  1. Provide the data:
    • Set L (cm):
      press A. Display = stored value of L.
      It is incorrect. Enter 200 and press R/S.
      Display = 0. End of procedure.

    • set H, B (cm):
      press B. Display = stored value of H (cm).
      It is incorrect. Enter 47 and press R/S.
      Display = stored value of B (cm).
      It is incorrect, then enter 30 and press R/S to store it.
      Display = 0. End of procedure.

    • set TMX, TMIN (dN):
      press C. Display = stored value of TMX (dN).
      It is incorrect. Enter 15000 and press R/S.
      Display = incorrect stored value of TMIN (dN).
      Enter the new value 8000 dN and press R/S to store it.
      Display = 0. End of procedure.

    • set ΣF, RCK (dN/cm²):
      press D. Display = incorrect stored value of ΣF.
      Enter the new value 1900 (dN/cm²) and press R/S.
      Display = incorrect stored value of RCK .
      Enter 250 (dN/cm²) and press R/S to store it.
      Display = 0. End of procedure.

    • set N, IP:
      press E. Display = incorrect stored value of N.
      Enter the new value 2 and press R/S.
      Display = incorrect stored value of IP .
      Enter 45 (degrees) and press R/S to store it.
      Display = 0. End of procedure.

    • set AP (cm²):
      press 2nd A'. Display = incorrect value of AP.
      Enter 6.03 (cm²) and press R/S to store it.
      Display = 0. End of procedure.

    • set DS (cm):
      press 2nd B'. Display = incorrect value of DS.
      Enter the new value 1 (cm) and press R/S to store it.
      Display = 0. End of procedure.

  2. Results:

    Pressing 2nd E'. the program display TAU = 11,82033097 dN/cm².

    Pressing R/S in sequence will display:

    %S= 70.20122285 %, ASM = 10.04496781 cm², PS=15.63764421 cm, and repeat the sequence.

    The program will also print the following report:
