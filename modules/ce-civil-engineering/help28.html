STR28 - STIFFNESSES 1

The program calculates the stiffness "K" of a beam as the ratio between the moment of inertia of the section with respect to its barycentric axis, and the length of the beam (K=J/L).

The stiffness is calculated for the sections:

  1. Rectangular, possibly also hollow;
  2. T-section;
  3. L-section;
  4. Circular section possibly hollow.
  5. U-section;

The section is considered constant for the whole length.

Any unit of measurement can be used.

The program uses the registers from 90 to 99 and therefore can be recalled during the elaboration of the structural programs, from STR22 to STR26, without interfering with their archives. It will be sufficient to transfer the value of the stiffness from the display to the structural program or others applications with the "Copy/Paste" function of the "Ti5x" App, illstrated in the instruction manual.

Method Of Calculation And Formulas Used

With reference to previous figure, called

L = length of the beam;

A = section area;

S = static moment of the section;

J = moment of inertia of the section;

X = neutral axis = S/A;

K = stiffness of the section = J/L;

the program calculates the stiffness using the following formulas:

  1. rectangular section, possibly also hollow:

    X= H/2; A= BH - BₒHₒ; J= (BH³- BₒHₒ³)/12;

  2. T-section:
    A= BₒH+(B-Bₒ)Hₒ; S= [BₒH²+(B-Bₒ)Hₒ²]/2;
    J= [BX³-(B-Bₒ)(X-Hₒ)³+B(H-X)³] / 3;
  3. L-section: the formulas coincide with the T-section;
  4. circular section possibly hollow:
    A= π(D²-Dₒ²)/4; J= π(D⁴- Dₒ⁴)/64;
  5. U-section:
    A= (B-Bₒ)H + Bₒ(H-Hₒ); S= [(B-Bₒ)H² + Bₒ(H-Hₒ)²]/2;
    J= [BX³+(B-Bₒ)(H-X)³ -B(X-H+Hₒ)³] / 3;

Example 1

With reference to the previous figure, calculates the stiffness of a hollow rectangular section beam having the following dimensions:

L= 400 cm; B=40 cm; Bₒ=20 cm; H=80 cm; Hₒ=60 cm.

The stored data is assumed to be zero. 1 - Provide the data:

  • Set L:

    press A. Display = 0 (stored value of L).

    Enter 400 and press R/S.

    Display = 0. End of procedure.

  • set B, Bo:

    press B. Display = 0 (stored value of B).

    It is incorrect. Enter 40 and press R/S.

    Display = 0 (stored value of Bo).

    It is incorrect, then enter 20 and press R/S to store it.

    Display = 0. End of procedure.

  • set H, Ho:

    press C. Display = 0 (stored value of H).

    It is incorrect. Enter 80 and press R/S.

    Display = 0 (stored value of Ho).

    It is incorrect. Enter 60 and press R/S to store it.

    Display = 0. End of procedure.
