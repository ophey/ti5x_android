STR-20 — Thrust of the Earth

The program identifies the pressure diagram and calculates the thrust of the earth against a retaining wall according to Rankine's hypothesis.

The units of measurement must be: lengths in meters, forces in Newton and related multiples or submultiples.

Provide the Data

  • F = internal friction angle of the ground (degrees)
  • G = specific weight of the ground (N/m³)
  • H = height of the wall above ground (m)
  • H'= underground height (m)
  • P = slope of the ground (degrees)
  • Q = load on the ground (N/m²)

Results:

  • B = angle of the ground breaking surface (degrees)
  • P1= pressure at the top of the wall (N/m²)
  • P2= pressure at the height H' of the wall (N/m²)
  • Z = resulting thrust on the height H of the wall (N)
  • E = height above ground of application of the thrust Z (m)
  • Z'= resulting thrust on the height H' of the wall (N)
  • E'= height of the application of thrust Z' from the base of the wall (m)

Instructions

The instructions for using the program are summarized in the "card emulation” similar to the magnetic card of the TI-59.

It summarizes the instructions assigned to each definable key, transcribed in the box referring to the key.

Each input procedure is independent and the operator can perform it without a predetermined order.

For another calculation, it will be sufficient to replace only the data that change.

With the procedure assigned to keys it is possible to reecall and correct the stored data. By pressing the key the data present in the register is extracted and displayed. Pressing the key again, the same data, possibly corrected, will be stored again.

Pressing 2nd E' the program calculates the thrust of the earth on the wall and displays "B".

Pressing in sequence R/S displays: P, P', Z, E, Z', E' and repeat the sequence.

The program will also print a report with data and results.

Example

Calculate the thrusts on the wall of the previous figure.

  1. Provide the Data:
    Each procedure can be performed without a specific order.
    • Set F : enter 20 degrees. Press A. Display = previous stored data.
    • Set G : enter 1400 dN/m³. Press B. Display = previous stored data.
    • Set H : enter 6 m. Press C. Display = previous stored data.
    • Set H': enter 1 m. Press D. Display = previous stored data.
    • Set E : enter 15 degrees. Press E. Display = previous stored data.
    • Set Q : enter 0 dN/m². Press 2nd A'.

  2. Results:

    Pressing 2nd E' the program calculates the thrust of the earth on the wall and displays B = 47.68014245 degrees.

    Pressing in sequence R/S displays: P, P', Z, E, Z', E'.

    The program will also print the following report with data and results

About the Program

Number of steps: from 000 to 332.

Registers used: from 00 to 17

Flags used: 1

Labels: Prt, Op, = , Adv, - , +, and definable keys.
