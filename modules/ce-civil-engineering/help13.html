STR13 - RECTANGULAR SECTION IN R.C.− BENDING PROJECT

The program calculates the stress in the concrete and area of the reinforcement of a rectangular reinforced concrete (r.c.) section, subject to bending, according to the method of admissible stresses. The original Texas Instruments program has been adapted to the higher performance of the "Ti5x" APP.

The units of measurement must be consistents and those of the International System: lengths in meters, forces in Newton and related multiples or submultiples.

The sign of the bending moment must always be positive (convention Science of the Constructions).

Instructions

1 - Provide the data:

Each input procedure is independent.
Enter data and press the definable key. The program stores the data and displays previously stored data.

  • enter M (bending moment), press A.
  • enter B (section width), press B.
  • enter A (section height), press C.
  • enter H’(steel cover thickness), press D.
  • enter G (homogenization coefficient), press E.
  • enter ΣFT (stress in tensioned irons), press 2ndA'.
  • enter P = AFC/AFT, press 2ndB'.

To view and correct a data, press its definable key twice. A first time to view it and correct it if necessary. A second time, to store it again in the same register.

2 - Results:

Pressing 2nd E', the program will display the neutral axis X.

Pressing in succession R/S will display the others results:

  • ΣC (stress in the concrete),
  • AFT (tensioned irons area),
  • AFC (compressed irons area).
  • For a new calculation, reset only the data that changes.

Example 1

Calculate the section in reinforced concrete having the following characteristics:

  • M=500000 dNcm;
  • B=30 cm
  • A=50 cm
  • H'=3 cm
  • G=10
  • ΣFT=1900 dN/cm²
  • P=(AFC/AFT)=0.2.

1 - Provide the data:

  • set M: enter 500000 dNcm, press A.
  • set B: enter 30 cm, press B.
  • set A: enter 70 cm, press C. (*) data incorrect.
  • set H': enter 3 cm, press D.
  • set G: enter 10, press E.
  • set ΣFT: enter 1900 dN/cm², press 2ndA'
  • set P: enter 0.2, press 2ndB'.

  • (*) To correct A: press C. Display = 70 (stored data),
    enter 50 cm, press C again to replace the previous incorrect data.
  • To display M: press A. Display = 500000 (correct data stored),
    press A to store it again.
2 - Results:
  • press 2nd E'. Display = 11.68415936 cm (neutral axis X)
  • pressing in succession R/S will display ΣC (dN/cm²), AFT, AFC (cm²) and repeat the sequence.
  • The program will also print the following report 
    
     --------------------
     STR13
           500000.      M
               30.      B
               50.      A
                3.     H'
               10.      G
             1900.    ∑FT
               0.2      P

       11.68415936      X
       62.86103451     ∑C
        6.09843403    AFT
       1.219686806    AFC
     --------------------
  • For a new calculation, reset only the data that changes.

About the Program

Number of steps: from 000 to 336.
Registers used: 00 to 12
Flags used: 1
Labels: Log, lnx, Adv, Op, - +, and definable keys.
